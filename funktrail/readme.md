# FunkTrail

Sito ufficiale del gruppo **FunkTrail**, realizzato con **Python Flask**.

## Struttura
- `app.py` — server Flask
- `templates/` — pagine HTML
- `static/` — CSS, immagini, foto, video, shop, contact

## Avvio locale
```bash
pip install -r requirements.txt
python app.py
